# 🚀 FundFlow NextGen Dashboard
**A Machine Learning-Powered Nonprofit Financial Risk Simulator for Aggie Hacks 2026**

FundFlow is a high-performance React dashboard designed to analyze and predict the financial resilience of over 11,500 nonprofit organizations. By ingesting tax return data (IRS Form 990) and applying advanced Machine Learning models, FundFlow instantly highlights systemic vulnerabilities and visualizes institutional health metrics to help stakeholders make data-driven funding decisions.

## 🌟 Key Features

* **💎 Hidden Gems Algorithm:** Instantly surfaces highly resilient, lean nonprofits that demonstrate strong programmatic autonomy but lack massive endowments.
* **⚠️ Critical Risk Tracking:** Identifies fragile organizations that are inherently dependent on isolated grant channels and simulates how they would withstand systemic funding shocks.
* **📈 7-Year Historical Trend Analysis:** Fully reactive, inline visualizations generated by Chart.js that map Total Revenue vs. Functional Expenses over nearly a decade of filings.
* **📊 Live Peer Benchmarking:** Displays z-score derived Performance Tiers (Top Performer, Above Average, Underperformer) and Income Diversity indices (HHI) for pinpointing specific systemic inefficiencies.

## ⚡ Tech Stack

* **Frontend:** React 18 & Vanilla JavaScript
* **Styling:** Custom CSS Glassmorphism with fully responsive grid layouts and dynamic UI scaling. 
* **Backend Pipeline:** Python (Data cleaning, ML classification parsing, and unification of disparate `csv` arrays into an optimized, memory-ready JSON state).
* **Compiler:** Babel Standalone for instantaneous browser rendering without a heavy Node.js environment.

## 🛠️ How to View Locally

Because the dashboard actively reads a massive 16MB locally-compiled `dataset.json` database, it requires a simple HTTP server to load dynamically. 

1. Open your terminal and navigate to this folder:
   ```bash
   cd fundflow-react
   ```
2. Start up a basic Python web server:
   ```bash
   python3 -m http.server 8000
   ```
3. Open your web browser and go to `http://localhost:8000`

## 🧠 Data Pipeline
The core data powering this dashboard originates from IRS 990 data forms that were aggressively cleaned and passed through a custom Random Forest architecture to produce `is_hidden_gem` logic, `resilience_score` arrays, and `shock_recovery_class` projections. 
The static `dataset.json` file feeding the frontend is procedurally generated by `generate_v2_dataset.py`, which normalizes the spreadsheet and nests historical tax filing years per individual EIN.
